@lichang666/utils ​

使用@lichang666/utils可以方便的进行开发。

安装 ​

推荐使用pnpm进行安装，以获得最佳性能。

bash
pnpm add @lichang666/utils

使用 ​

在ts文件中使用import引入即可。

ts
import { getRandomColor } from '@lichang666/utils'

console.log(getRandomColor()) // 随机颜色
